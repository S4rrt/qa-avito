### 1. Нарушение фильтра по городу

**Описание:**  
В выдаче "Котята в Москве" присутствует объявление из города Новосибирск.

**Приоритет**

P0 - Критическое

Ошибка бизнес-логики. Нарушает основной сценарий - поиск. 
Пользователь получает неправильный результат

**Ожидаемый результат:**  
Все объявления соответствуют городу Москвы.

---

### 2. Некорректная работа фильтра по породе

**Описание:**  
При выбранной породе "Персидская" в выдаче присутствуют объявление породы "Экзотический короткошёрстный".

**Приоритет:**  
P0 - Критическое

Ошибка бизнес-логики. 
Фильтр не выполняет основную функцию - сужение поиска по породе кошки.  
Пользователь получает неправильный результат.

**Ожидаемый результат:**  
Выдача результата строго соответствует выбранной породе "Персидская".

---

### 3. Некорректная работа фильтра по цене

**Описание:**  
При выбранной диапазоне цен от 1000 до 30000 рублей в выдаче 
присутствует объявления с ценой 500 рублей

**Приоритет**  
P0 - Критический 

Ошибка бизнес-логики. Фильтр не выполняет основную функцию - сужение поиска по цене.
Пользователь получает неправильный результат.

**Ожидаемый результат:**  
Корректные и валидные значения цен либо дополнительная модерация.


### 4. Несоответствие текста при наведении

**Описание:**  
Иконка сердца при наведении показывает текст  
"Не показывайте мне похожие объявления".

**Приоритет**  
P2 - Средний

Нарушена UX - текст подсказки не соответствует ожидаемому поведению.  
Это может привести к заблуждению пользователя.

**Ожидаемый результат:**
Иконка сердца при наведении показывает текст "Добавить в избранное"

---

### 5. Ошибка текста в фильтре "Возраст" 

**Описание:**  
При поиске "Кошки" в фильтре "Возраст" написан "Щенок"

**Приоритет**  
P2 - Средний

Нарушена UX - ошибка текста вводит пользователя в заблуждения.

**Ожидаемы результат**  
При поиске "Кошки", фильтр "Возраст" показывает возрастные категории "Любой","Котёнок","Взрослая"
