Склонируйте к себе репозиторий, в котором хранится проект тестового задания, через выполнение команды в терминале
```
git clone https://github.com/S4rrt/qa-avito.git
```
Или скачайте zip-архив по ссылке и распакуйте его  
Убедитесь, что на вашем компьютере установлен Go. В командной строке выполните команду:

```
go version
```

Если он не установлен, скачайте его с официального сайта https://go.dev/dl/, выбрав подходящую версию для вашей операционной системы.

Через командную строку перейдите в корневую директорию проекта:

```
cd /здесь/укажите/путь/до/директории/с/проектом
```

Установите необходимые зависимости:

```
go mod tidy
```

Запустите тесты:

```
go test ./... -v -timeout 60s
```
